## E-commerce Gtech ##
Pensando em resolver um problema que surge na hora de comprar um novo Computador, decidimos criar esse projeto que foca no auxilio para escolher as peças do PC.
O site oferece um Chatbot inteligente que aceita mensagens de texto e imagens para auxiliar o cliente, neste chat o cliente irá poder fazer suas perguntas e no final da conversa, ter certeza que está comprando as peças compativeis entre sí.

## Informações para executar o projeto em sua maquina! ##
    - versão do Python utilizada: Python 3.11.1
    - Deve-se criar e ativar o ambiente virtual antes de executar o programa!
    - Os comando para executar essa tarefa são:
    - Crindo o ambiente virtual -> python -m venv venv
    - Ativando o ambiente virtual -> venv/scripts/activate
    - #Após fazer-mos as instruções acima, devemos instalar as bibliotecas utilizadas no projeto.
    - #Executamos o comando -> pip install -r requirements.txt
